#  Sentiment Analysis

## Objective
To analyze textual data and determine sentiment polarity using basic natural
language processing techniques.

## Dataset
- Sample text data representing user feedback or social media comments
- Each text entry is analyzed to determine sentiment

## Methodology
- Created a text dataset for sentiment analysis
- Used TextBlob to calculate sentiment polarity scores
- Classified text into Positive, Negative, or Neutral categories
- Visualized sentiment distribution using bar charts

## Sentiment Analysis Technique
- Library Used: TextBlob
- Sentiment Metric: Polarity score
- Classification Logic:
  - Positive sentiment: polarity > 0
  - Negative sentiment: polarity < 0
  - Neutral sentiment: polarity = 0

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- TextBlob
- Google Colab

## Results
- The sentiment distribution shows both positive and negative opinions
- Positive sentiments slightly dominate the dataset

## Conclusion
This task demonstrates how basic sentiment analysis can be applied to textual
data to extract meaningful insights using simple NLP techniques.
